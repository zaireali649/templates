# Learning Accelerator

Master new topics quickly with structured learning.

---

## Prompt

```
Teach me [topic].

Structure:
1. Explain like I'm 5
2. Explain at professional level
3. Real-world example
4. Hands-on mini project
5. Common mistakes to avoid
6. 3 key takeaways
```

---

## Example Usage

```
Teach me Kubernetes.

Structure:
1. Explain like I'm 5
2. Explain at professional level
3. Real-world example
4. Hands-on mini project
5. Common mistakes to avoid
6. 3 key takeaways
```

---

## What You'll Get

**ELI5**: Simple metaphor (Kubernetes is like a smart warehouse manager...)
**Professional**: Technical details, architecture, components
**Real-world**: How Netflix/Spotify use it
**Mini project**: Deploy a simple app to Kubernetes
**Mistakes**: Don't set resource limits too low, don't ignore health checks
**Takeaways**: 3 core concepts to remember

---

## Why This Works

- **ELI5**: Builds intuition
- **Professional**: Fills in technical depth
- **Example**: Shows practical application
- **Project**: Learning by doing
- **Mistakes**: Avoids common pitfalls
- **Takeaways**: Reinforces core concepts

---

## When to Use

- Learning new technology
- Need to understand a concept quickly
- Preparing for interview
- Evaluating whether to adopt a tool
- Teaching others
