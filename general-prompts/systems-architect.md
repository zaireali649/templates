# Systems Architect

Design production-ready systems with AI.

---

## Prompt

```
Act as a senior systems architect.

Goal: design a production-ready solution for [describe system].

Include:
- architecture diagram (text-based)
- components and responsibilities
- tradeoffs and alternatives considered
- scaling plan
- risks and mitigations
- step-by-step implementation plan
```

---

## Example Usage

```
Act as a senior systems architect.

Goal: design a production-ready solution for a real-time analytics dashboard 
processing 100K events per second from IoT devices.

Include:
- architecture diagram (text-based)
- components and responsibilities
- tradeoffs and alternatives considered
- scaling plan
- risks and mitigations
- step-by-step implementation plan
```

---

## What You'll Get

**Architecture diagram** in text format
**Components**: Message queue, stream processor, time-series DB, API, frontend
**Tradeoffs**: Kafka vs Kinesis, ClickHouse vs TimescaleDB
**Scaling**: Horizontal scaling strategy, data partitioning
**Risks**: Data loss, latency spikes, cost overruns
**Implementation**: Phased rollout with milestones

---

## When to Use

- Starting a new system
- Evaluating architectural approaches
- Need to explain architecture to stakeholders
- Scaling an existing system
- Making build vs buy decisions
