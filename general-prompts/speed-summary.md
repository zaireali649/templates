# Speed Summary

Get to the point fast.

---

## Prompt

```
Summarize this into:
- 5 bullet points (key information)
- 1 "so what" takeaway (why it matters)

[paste content]
```

---

## Example Usage

```
Summarize this into:
- 5 bullet points (key information)
- 1 "so what" takeaway (why it matters)

[paste 10-page technical document]
```

---

## What You'll Get

**5 Bullets**:
- Most important facts
- Key data points
- Critical decisions
- Main recommendations
- Notable changes

**So What**:
One sentence explaining why this matters and what to do about it.

---

## When to Use

- Long documents/articles
- Meeting notes
- Research papers
- Email threads
- Technical reports

---

## Variations

**Ultra-fast version**:
```
TL;DR in one sentence:
```

**Detailed version**:
```
Summarize into:
- 10 bullet points
- Key takeaways
- Action items
- Questions to resolve
```
